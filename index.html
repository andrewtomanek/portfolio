Zdravá strava

Kalkulátor výživových hodnot vložených potravin.
Po nastavení denního limitu se vypočítá v košíku suma přidaných položek. Výsledek se zobrazí graficky a v procentech.
Po autorizaci je možné osobní data ukládat a stahovat z databáze Firebase.

  • Single page app pohání knihovna React.js.
  • State managment na bázi Redux.
  • Asynchronus IO Redux-saga.
  • Animace přes react-transition-group.
  • CRUD operations s pomocí Axios.
  • Database and Authorization s využitím Google Firebase
  • Testing coverage handled with Jest and Enzyme.
  • Type checking by TypeScript.
  • Styling elments with styled-components.

Pro Input/Output funkčnost je potřeba vytvořit ůčet u Google Firebase a uložit API keys v .env souboru ve formátu: const API_KEY = 'ENTER YOUR API'

Dependencies:
React.js Redux.js React-Router Redux-saga react-transition-group Axios Firebase styled-components Jest Enzyme TypeScript

E-shop Node.js

Internetový obchod s vínem.
Řazení položek podle ceny nebo abecedy vzestupně nebo sestupně.
Modifikace položek v košíku zadaním do input field nebo tlačitek +/-.

  • Server rendered HTML
  • View engine EJS.
  • Databáze MongoDb cloud.
  • Knihovna Mongoose pro snadnou obsluhu ModgoDb.

Node Express EJS MongoDb cloud Mongoose Nodemailer
